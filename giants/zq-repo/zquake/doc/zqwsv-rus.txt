1. О проекте ZQuake (точнее, о zquake-сервере)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
ZQuake-сервер (zqwsv) - это в принципе обычный QW-сервер с небольшими
дополнениями.  Самое приятное из них --- возможность играть на локальном
сервере с пингом 14, и при этом не требуется запускать priority.exe!
Кроме того, в zqwsv нормально работают распрыги (но при желании это можно
отключить, поставив pm_jumpfix в 0).
И, наконец, на zqwsv можно играть с монстрами (если поставить специальный
мод, который можно скачать с zquake.frag.ru).


1. Список новых команд
~~~~~~~~~~~~~~~~~~~~~~
* cmdlist - вывести список всех команд
* cvarlist - вывести список переменных
* toggle - инвертировать значение переменной (1 -> 0 и 0 -> 1)
* unalias - уничтожить существующий alias
* set - установить/создать переменную
* inc - увеличить значение переменной на 1, или (если указан второй параметр)
	прибавить к переменной произвольное число.


2. Список новых переменных (в скобках - значение по умолчанию)
~~~~~~~~~~~~~~~~~~~~~~~~~~

* pm_jumpfix (1) - фикс для прыжкового бага.

* pm_ktphysics (0) - ммм не рекомендую крутить эту переменную :)
  Она устанавливает такую же физику прыжка с наклонной плоскости, как под
  модом Kombat Teams, и используется в основном в клиенте (zqwcl) для игры
  на серверах Kombat Teams, а на сервере ее ставить в 1 не имеет смысла.

* pm_slidefix (0) - при спуске с наклонной поверхности игрок не "сваливается"
  (баг QW), а плавно "съезжает" (как было в NQ).

* sys_sleep (8 на Win9x, 0 на WinNT) - число миллисекунд, на которые передавать
  управление другим процессам за один серверный цикл.  Не рекомендую играть
  с этой переменной --- машину, может, и не повесите, но тормозить может
  начать конкретно :)

* coop (0) - режим "кооператив" (для single player)
* skill (1) - уровень сложности (для single player)


3. Priority fix
~~~~~~~~~~~~~~~
При запуске zqwsv устанавливает сам себе высокий приоритет
(HIGH_PRIORITY_CLASS).  Это необходимо для того, чтобы при игре на локальном
сервере под Windows 98 не возникал большой PL (порой доходящий до 50).
Если вы по какой-либо причине хотите отключить автоматическую установку
приоритета, запускайте сервер с параметром -nopriority:
zqwsv -nopriority.
