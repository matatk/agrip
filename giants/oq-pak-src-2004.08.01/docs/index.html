Open Quartz - pak0 module

Open Quartz is a project to supply GPL artwork to create a fully GPL game based around the GPL Quake sourcecode. Click here for information about contributing. The pak0 module contains content (maps, models, graphics and sounds) and Quake-C code which controls the behavior and rules of the game.

  • General Public License
  • Menu and Status Bar Graphics
  • Quake-C game logic, AI and rules
    • "Normal Quake" (NQ) compatible code with Open Quartz messages
    • "QuakeWorld" (QW) compatible code with Open Quartz messages
    • Merged (NQ and QW) code (uses C compiler for pre-processing)
